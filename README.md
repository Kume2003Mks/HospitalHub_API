## Installation

```bash
npm install
```

## Running the app

```bash
# development
npm run start

# watch mode
npm run dev

# production mode
npm run start:prod
```

## Test

```bash
# unit tests
npm run test

# e2e tests
npm run test:e2e

# test coverage
npm run test:cov
```

## Run In Docker 

```bash

#build command
docker build -t hospitalhub-nestjs .

# run in net work
docker run -d --name nestjs-app -p 3001:3001 hospitalhub-nestjs
```
